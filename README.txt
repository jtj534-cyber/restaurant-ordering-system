Program Description:
This project is a Restaurant Ordering System developed in Python. The program allows the user to display a digital menu, add items to a basket, update quantities if an item already exists, remove items from the basket, and sort menu items by price. It also calculates the subtotal, tax, and final total for each order. The checkout function includes payment validation to ensure the payment is sufficient before the order is completed.

The program also saves completed orders to a JSON file and generates a daily sales report by reading saved order data. Error handling is included to prevent crashes when invalid input is entered, such as incorrect menu IDs, negative quantities, or invalid payment values. The project demonstrates the use of functions, lists, dictionaries, file handling, sorting, calculations, and validation in a practical way. The program was designed as an intermediate-level project focused on structured logic, clear functionality, and robust user interaction.
