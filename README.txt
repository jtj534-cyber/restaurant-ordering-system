Program Description:
This project is a Restaurant Ordering System developed in Python. The program allows the user to view a digital menu, add items to a basket, update quantities, remove items, and sort menu items by price. It also calculates the subtotal, tax, and final total for the order. The system includes payment validation to make sure the payment is enough before completing checkout.

In addition, the program saves each completed order to a JSON file and can generate a daily sales report by reading saved order data. Error handling is included to prevent the program from crashing when invalid input is entered. For example, the program checks for invalid menu IDs, negative quantities, and incorrect payment values. This project was designed to show the use of functions, lists, dictionaries, file handling, sorting, calculations, and validation in a clear and practical way.
